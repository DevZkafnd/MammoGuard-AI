⚠ PENTING: Lakukan ini terlebih dahulu!

Cara Restart Docker Desktop:
1. Klik kanan icon Docker di system tray (pojok kanan bawah)
2. Pilih "Quit Docker Desktop"
3. Tunggu 15 detik sampai benar-benar tertutup
4. Buka Docker Desktop lagi dari Start Menu
5. Tunggu sampai status "Docker Desktop is running"
6. PENTING: Tunggu 30 detik lagi! (Biar Docker API ready)
